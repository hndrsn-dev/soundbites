Spacing

Base-4 grid. Prefer multiples of 8 for padding and gaps.
Rule

All layout spacing — padding, margin, gap — must be a multiple of 4px. Prefer multiples of 8px wherever possible. Use 4px only in dense/compact contexts (icon padding, tight inline gaps).

Exceptions: 1px–2px values are acceptable for borders, hairlines, and purely decorative pixel elements (waveform bar heights, indicator dot sizes). These are not layout spacing and are not subject to the grid.
